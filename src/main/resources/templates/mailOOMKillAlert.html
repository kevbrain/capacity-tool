It looks like you have one or more pods with high memory consumption causing that pod(s) to restart

<#list appNames as app>
Applications
${app}

It's recommended to increase the memory. http://wiki-it/display/DEVOPS/OCP+FAQ+and+Troubleshooting#OCPFAQandTroubleshooting-Exceedingquotas,limitedresources
You just have to declare the request and limits placeholders in your XLD dictionary placed before (order of the dictionaries) the dico openshift-ocnp1-dict
Find below the list of placeholders to overwrite :
For the ms, please increase the following values in XLD

<#list pods as pod>
${pod.namespace.name}
XLD PlaceholdersCurrent Value
project.memory.request${pod.pod.requests_memory}
project.memory.limit${pod.pod.limits_memory}


Take care ! if we don't do this, at each deployment the values will be set with the values of the default dictionary.
If you have any issue with quota, don't hesitate to contact us.
